(constant) resolverUrl

Source:
  • resolver.js, line 10

module resolver

Interfaces with the holo resolver to locate the DNA or host tranche for a given URL. This is not required in most cases where the UI is also hosted by Holo but in some cases where the UI is hosted on the conventional web this is required to locate hosts.
