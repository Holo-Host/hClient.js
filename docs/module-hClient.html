Module: hClient

hClient The javascript library for making your web UI Holo enabled! It handles: Key management and generation, Generating temporary readonly keys for browsing, Detecting when authorization is required and prompting the user to signup/login to generate read/write keys, Signing calls and responses, Setting up a websocket connection to the interceptor to sign commits on request, Wrapping and unwrapping calls to and from the interceptor such that they look like regular holochain calls
